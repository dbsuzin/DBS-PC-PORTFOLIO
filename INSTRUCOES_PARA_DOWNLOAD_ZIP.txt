=== COMO USAR ===

1. Baixe o arquivo dbs-pc-portfolio-completo-2026-07-10.zip

2. Descompacte em uma pasta NOVA (ex: D:\Projetos\DBS-PC-PORTFOLIO-NOVO)

3. Entre na pasta descompactada e rode (PowerShell ou CMD):
   npm install

4. Configure seu .env.local (se necessário para teste local):
   DATABASE_URL="sua-url-do-neon"

5. Faça deploy no Vercel:
   - Conecte a pasta no Vercel (ou faça git push)
   - O build vai usar automaticamente os comandos do vercel.json

6. Depois que estiver no ar, rode o agente novamente no seu PC:
   powershell -ExecutionPolicy Bypass -File ".\scripts\agent.ps1" -ApiKey "SUA_CHAVE"

7. No site faça hard refresh (Ctrl + Shift + R)

Pronto! O erro de npm install + Prisma no Vercel deve estar resolvido.

Se ainda der erro, me manda o log completo do deploy.
