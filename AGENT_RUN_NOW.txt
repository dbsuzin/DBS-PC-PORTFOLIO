================================================================================
           COMANDO EXATO PARA RODAR O AGENTE ATUALIZADO (2026-07-10)
================================================================================

1. COPIE o arquivo scripts/agent.ps1 atualizado para o seu PC (ou baixe do projeto).

2. Abra o **PowerShell como Administrador**.

3. Rode o comando abaixo (substitua SUA_API_KEY pela chave da empresa):

powershell -ExecutionPolicy Bypass -File ".\agent.ps1" -ApiKey "SUA_API_KEY_AQUI"

================================================================================
EXEMPLO COMPLETO (use sua chave real):
================================================================================

powershell -ExecutionPolicy Bypass -File ".\agent.ps1" -ApiKey "cmre532k50001q4my81fh5uk2"

================================================================================
O QUE FOI MELHORADO NO AGENTE:
================================================================================
- Múltiplos fallbacks para osInstallDate e lastBootTime
- Força envio como string ISO (formato "o")
- Logs de DEBUG mostrando o que está sendo enviado
- Upsert no servidor (atualiza PC em vez de duplicar)
- Mensagens mais claras

Após rodar, você verá:
  [ENVIANDO] osInstallDate: ...
  [ENVIANDO] lastBootTime: ...

Volte no site e atualize (F5) - as colunas "Inst. SO" e "Últ. Boot" devem aparecer.

Se ainda aparecer "—", cole a saída completa do PowerShell aqui para diagnóstico.
