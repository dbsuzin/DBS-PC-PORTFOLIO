================================================================================
           COMANDO EXATO PARA RODAR AGORA
================================================================================

1. Delete o arquivo atual agent.ps1

2. Baixe novamente o arquivo scripts/agent.ps1 do projeto

3. Abra o PowerShell **como Administrador**

4. Rode **exatamente** este comando:

powershell -ExecutionPolicy Bypass -File ".\agent.ps1" -ApiKey "cmre532k50001q4my81fh5uk2"

================================================================================
SE AINDA DER ERRO DE SINTAXE
================================================================================

Use este comando alternativo (mais seguro):

powershell -ExecutionPolicy Bypass -Command "& { .\agent.ps1 -ApiKey 'cmre532k50001q4my81fh5uk2' }"

Ou rode direto:

powershell -ExecutionPolicy Bypass -Command ".\agent.ps1 -ApiKey 'cmre532k50001q4my81fh5uk2'"
