================================================================================
           SOLUÇÃO RÁPIDA - Erro de "não está assinado digitalmente"
================================================================================

O erro acontece porque o Windows bloqueia scripts .ps1 por segurança.

COMANDO QUE VOCÊ DEVE RODAR AGORA (copie e cole):

Abra o PowerShell **como Administrador** e cole este comando completo:

powershell -ExecutionPolicy Bypass -File ".\agent.ps1" -ApiKey "cmre532k50001q4my81fh5uk2"

Ou, se preferir em uma linha só:

powershell -ExecutionPolicy Bypass -Command ".\agent.ps1 -ApiKey 'cmre532k50001q4my81fh5uk2'"

================================================================================
DEPOIS QUE FUNCIONAR
================================================================================

Se quiser que nunca mais dê esse erro, rode uma vez como Administrador:

Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

Depois disso você pode rodar normalmente:
.\agent.ps1 -ApiKey "cmre532k50001q4my81fh5uk2"

================================================================================
PARA AGENDAR NO AGENDADOR DE TAREFAS (recomendado)
================================================================================

No Agendador de Tarefas use estes argumentos:

-ExecutionPolicy Bypass -File "C:\Script_dbs\agent.ps1" -ApiKey "cmre532k50001q4my81fh5uk2"
